## 🛠️ Tech Stack

### Frontend
- **React 18** with Vite
- **Tailwind CSS** for styling
- **React Router DOM** for navigation
- **Lucide React** for icons

### Backend
- **Java Spring Boot**
- **Spring Data JPA**
- **MySQL/PostgreSQL** database

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- Java 17+
- MySQL/PostgreSQL

### Development Setup
```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend (coming soon)
cd backend
./mvnw spring-boot:run
```
